Desafio 1

Crie uma página da web em HTML com um botão que, quando clicado, exibe um alerta com uma mensagem simples. Implemente isso usando um ouvinte de eventos.

Desafio 2

Construa uma página com uma imagem. Quando um usuário passar o ponteiro do mouse sobre a imagem, altere a origem da imagem para outra imagem. Use ouvintes de eventos para realizar essa tarefa.

Desafio 3

Desenvolva uma aplicação de lista de tarefas simples. Os usuários devem ser capazes de adicionar tarefas digitando em um campo de entrada e pressionando a tecla "Enter". Quando as tarefas forem adicionadas, elas devem aparecer em uma lista na página da web, usando ouvintes de eventos para lidar com o evento de pressionar tecla e atualizar o DOM.

Desafio 4

Tarefa: Crie uma galeria de imagens com miniaturas e uma imagem principal. Quando um usuário clicar em uma miniatura, a imagem principal deve mudar para exibir a imagem selecionada. Implemente essa funcionalidade usando ouvintes de eventos e JavaScript.

Desafio 5

Construa uma aplicação de chat em tempo real. Os usuários devem ser capazes de enviar e receber mensagens instantaneamente. Utilize ouvintes de eventos para a entrada e exibição de mensagens. Considere o uso de tecnologias como WebSockets para obter funcionalidades em tempo real.

Corretoras

Aplicação de Cookies e Sessões

O projeto de Lista de Tarefas com Sistema de Login é uma aplicação web desenvolvida para ajudar os usuários a organizar suas tarefas de maneira eficiente. Além da funcionalidade principal de criar, editar e excluir tarefas, o sistema possui um sistema de autenticação robusto baseado em cookies e sessões para garantir a segurança e a privacidade dos usuários.
